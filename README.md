📱 do_frontend

사내 업무 효율 증진 및 구성원 관리를 위한 애플리케이션

🧩 프로젝트 소개

do_frontend는 사내 구성원의 업무 효율을 높이고,
조직 내 정보 관리 및 커뮤니케이션을 한곳에서 처리할 수 있도록 하는 사내 관리 애플리케이션입니다.
본 프로젝트에서 Flutter 기반 프론트엔드 개발을 담당했습니다.

🚀 주요 역할 (Frontend)

Flutter 기반 UI 개발 및 화면 구성

재사용 가능한 UI 컴포넌트 구현

Backend API와의 연동을 통해 주요 기능 구현
(로그인, 사용자 정보 조회, 데이터 표시 등)

간단한 상태관리 및 기본 로직 처리

🛠 기술 스택

Framework: Flutter

Language: Dart

API: REST API 기반 연동

📁 프로젝트 구조

(실제 폴더 구조에 맞게 수정 가능)

lib/
 ├── components/     # 재사용 UI 컴포넌트
 ├── screens/        # 주요 화면 페이지
 ├── services/       # API 연동 클래스
 ├── models/         # 데이터 모델 정의
 └── main.dart       # 앱 진입점

📱 주요 기능

로그인/회원 인증

구성원 정보 조회

공지/업무 관련 데이터 확인

기본적인 리스트/상세 UI 구현
