## 📱 do_frontend: 사내 업무 효율 증진 및 구성원 관리 애플리케이션

### 🧩 프로젝트 소개

`do_frontend`는 **사내 구성원의 업무 효율을 높이고, 조직 내 정보 관리 및 커뮤니케이션을 한곳에서 처리**할 수 있도록 설계된 사내 관리 애플리케이션의 프론트엔드 프로젝트입니다.

본 프로젝트는 **Flutter**를 기반으로 개발되어, 효율적인 UI/UX와 크로스 플랫폼 호환성을 제공합니다.

### 🚀 주요 역할 (Frontend)

| 카테고리 | 상세 역할 |
| :--- | :--- |
| **UI/UX 개발** | Flutter 기반 UI 개발 및 반응형 화면 구성 |
| **컴포넌트** | 재사용 가능한 UI 컴포넌트(위젯) 구현 및 관리 |
| **API 연동** | Backend REST API와의 연동을 통한 주요 기능 구현 (로그인, 사용자 정보 조회, 데이터 표시 등) |
| **로직 처리** | 간단한 상태관리 및 화면별 기본 비즈니스 로직 처리 |

### 🛠 기술 스택

| 분류 | 기술 | 설명 |
| :--- | :--- | :--- |
| **프레임워크** | **Flutter** | 크로스 플랫폼(iOS, Android 등) 앱 개발 프레임워크 |
| **언어** | **Dart** | Flutter 개발에 사용되는 프로그래밍 언어 |
| **API 통신** | **REST API** | 백엔드 서버와의 데이터 연동 방식 |

### 🔑 주요 기능

* **로그인/회원 인증**: 사내 구성원 인증 및 접근 관리
* **구성원 정보 조회**: 조직 구성원의 기본 정보 및 연락처 조회
* **공지/업무 데이터 확인**: 회사 공지사항 및 업무 관련 데이터 리스트 확인
* **기본적인 리스트/상세 UI 구현**: 데이터의 목록 보기 및 상세 내용 확인 기능 제공
